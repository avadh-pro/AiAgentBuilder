**Theme:** Write the full draft. Don't polish today. Just get every tweet to first-draft prose.

## Today's actions

### 1. Pinned post first draft (3 hours, single focused block)
Take each of yesterday's 12 claim-lines. Write the full tweet underneath each. Constraints:

- Each tweet must **stand alone** — someone could quote any one of them in isolation and it should still make sense
- Cut every word that isn't earning its space
- Use **specific examples**, not abstract principles. ("Last week an agent I shipped failed when X happened" beats "agents fail when context shifts" — reframe generically; never name your employer)
- Add `[image: <description>]` placeholders where a diagram or screenshot would carry weight
- Stay under 280 chars per tweet

After drafting, **read the full thread out loud once.** Two checks:
1. If any tweet makes you flinch when you read it aloud, mark it with `// FIX` for tomorrow
2. If the thread loses its through-line at any point, mark with `// BREAK` — that's where the argument needs another tweet or a smoother transition
